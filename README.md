# Dev Instructions
## Run backend

1. Open XAMPP Apache and MySQL
2. `cd scopecliq-app`
3. `composer install`
4. `php artisan serve`
5. `php artisan migrate:fresh --seed`
6. Open browser at localhost:8000

## Run frontend
1. `cd scopecliq-ui`
2. `npm run start`
6. Open browser at localhost:3000
